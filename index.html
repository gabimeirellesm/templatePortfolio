My projects

Scary Game

This project was my first web development project ever, I developed all the functionalities and styling in 4 days. It's a scary game built with Javascript, HTML + Canvas, CSS, and deployed on GitHub. The objective of the game is to take a picture of the ghost to provide evidence to the police. I encourage you to play this game and read more about its story

Trace Your Road

The main purpose of this project was to build a full-stack application that runs on a server using all the CRUD actions. We developed a desktop app in 4 days where users can write comments and choose all the countries they have visited. The app also counts the total percentage of the world visited and uses an API to automatically fill some basic information about each country.

  • Learn More

Drink Generator

The purpose of this mobile app project was to build a full-stack application with all the CRUD operations using React JS. The Drink Generator app allows users to filter and discover new cocktails recipes, as well as view articles related to drinks. Users can also create and save their own drink and article or select a random cocktail option.
